  1. BFF: backend for frontend 服务于后端的前端,也就是服务器设计 API 时会考虑前端的使用，并在服务端直接进行业务逻辑的处理，又称为用户体验适配器。BFF 只是一种逻辑分层，而非一种技术，虽然 BFF 是一个新名词，但它的理念由来已久.

  2. Docker:是一个开源的应用容器引擎, 相似概念有虚拟机
